# A1 워크시트 — 허수부 = 2-형식 (책 7장에서 풀 것)

> **목표.** "FS **폼**은 허미션 내적의 허수부에 붙는 이름이고, QFIM은 그 실수부다 — 계수 데이터는 같다"를 손계산으로 소유한다. 문제 원문은 책 7장(②G·H)에서 그대로 가져왔고([W]/[★]/[★★] 태그·검산용 답 포함), 마지막 세 문제(A1-1~3)는 이번 피드백을 과제로 옮긴 것이다(편집자 추가).
> **경로:** 준비운동 G1–G4 → 본운동 G5 → 다리 H3·H4 → 본운동 H5·H6 → A1-1·2·3. 규칙은 늘 그대로: [W] 하루 1개 배급제, 네 줄 기록(대상/계산/검산/번역), 막힌 [★]는 경계 기록.
> **검산 스크립트:** `verifyA1_targets.py` — **다 푼 뒤에** 돌릴 것. 책에 인쇄된 검산용 답과 기준값만 확인한다 (풀이 없음).

## 필요한 정의 (원문)

$$\alpha = \frac{i}{2} \sum_{r=1}^{2} \left( z_r d\overline{z}_r - \overline{z}_r dz_r \right)$$

이고 실좌표로는

$$\alpha = \sum_{r=1}^{2} (x_r dy_r - y_r dx_r).$$

Reeb vector field는

$$\alpha(R_\alpha) = 1, \qquad \iota_{R_\alpha} d\alpha = 0$$

를 만족하는 vector field이다.

---

## 준비운동 — α를 손에 넣기

### [★] ②G1. 복소식에서 실좌표식 얻기

하나의 복소좌표 $z = x + iy$에 대해

$$\frac{i}{2}(z d\overline{z} - \overline{z} dz)$$

를 계산하라. 결과가 $x\,dy - y\,dx$ 임을 보여라. 따라서 $\alpha = \sum_{r=1}^{2} (x_r dy_r - y_r dx_r)$ 이다.

### [★] ②G2. Fiber 위에서 α 계산하기

fiber 하나를 $\theta \mapsto (e^{i\theta}, 0)$ 으로 잡는다. 즉 $z_1 = e^{i\theta},\ z_2 = 0$, 실좌표로는 $x_1 = \cos\theta,\ y_1 = \sin\theta$.

$$\alpha = x_1 dy_1 - y_1 dx_1$$

를 계산하여 $\alpha = d\theta$ 임을 보여라.

### [W] ②G3. Reeb 후보 vector field 쓰기

다음 vector field를 생각한다.

$$R_\alpha = \sum_{r=1}^{2} \left( -y_r \partial_{x_r} + x_r \partial_{y_r} \right).$$

이것이 복소좌표에서는 $(z_1, z_2) \mapsto (e^{i\theta} z_1, e^{i\theta} z_2)$ 의 infinitesimal generator임을 확인하라.

### [W] ②G4. α(R_α) = 1 확인하기

$\alpha = \sum_r (x_r dy_r - y_r dx_r)$ 와 G3의 $R_\alpha$를 이용하여 $\alpha(R_\alpha)$ 를 계산하라.

힌트: $dx_r(R_\alpha) = -y_r,\ dy_r(R_\alpha) = x_r$.
결과: $\alpha(R_\alpha) = x_1^2 + y_1^2 + x_2^2 + y_2^2 = 1$ on $S^3$.

## 본운동 1 — d를 취하면 2-형식이 태어난다

### [★★] ②G5. dα 계산하기

$$\alpha = \sum_{r=1}^{2} (x_r dy_r - y_r dx_r)$$

에서 $d\alpha$ 를 계산하라.

검산용 답: $d\alpha = 2\sum_{r=1}^{2} dx_r \wedge dy_r$.

---

## 다리 — connection에서 curvature로 (주제 7 겸용)

### 필요한 정의 (원문)

$\mathbb{C}P^1$의 chart $U_0 = \{[z_1 : z_2] : z_1 \neq 0\}$ 에서 $w = z_2/z_1$ 라고 둔다. local section은

$$s_0(w) = \frac{(1, w)}{\sqrt{1 + |w|^2}}$$

로 잡는다.

### [★★] ②H3. s₀*α 계산하기

$$s_0^*\alpha$$ 를 계산하라.

검산용 답: $s_0^*\alpha = \dfrac{i}{2}\dfrac{w d\overline{w} - \overline{w} dw}{1 + |w|^2}$. 이것을 local connection form이라고 볼 수 있다.

### [★★] ②H4. Curvature 계산하기

$A = s_0^*\alpha$ 라고 두자. $dA$ 를 계산하라.

검산용 답: $dA = \dfrac{i\, dw \wedge d\overline{w}}{(1 + |w|^2)^2}$. 따라서 $\omega_{FS} = \dfrac{i\, dw \wedge d\overline{w}}{(1 + |w|^2)^2}$ 로 두면 $d\alpha = \pi^*\omega_{FS}$.

---

## 본운동 2 — 그 2-형식을 실좌표로, 그리고 적분

### [★] ②H5. 실좌표로 ω_FS 쓰기

$w = x + iy$ 라고 하자. $dw \wedge d\overline{w}$ 를 계산하라.

검산용 답: $dw \wedge d\overline{w} = -2i\, dx \wedge dy$, 따라서 $i\, dw \wedge d\overline{w} = 2\,dx \wedge dy$.

### [★] ②H6. ∫ ω_FS = 2π 계산하기

H5의 식을 이용해

$$\int_{\mathbb{C}} \frac{2\,dx\,dy}{(1 + x^2 + y^2)^2}
=\int_0^{2\pi}\!\!\int_0^{\infty} \frac{2r}{(1 + r^2)^2}\, dr\, d\theta$$

를 극좌표로 계산하라.

검산용 답: $\int_{\mathbb{C}P^1} \omega_{FS} = 2\pi$.

## A1 고유 과제 — 피드백을 과제로 (편집자 추가)

### [★] A1-1. 같은 벡터 두 개를 넣으면 허수부가 죽는다 — 그래서 2-형식이다

(a) 내적의 정의만 사용해서: 임의의 허미션 내적에 같은 벡터를 두 번 넣으면 값이 $\|v\|^2$ (음이 아닌 실수)여야 함을 쓰고, $\mathbb{C}^2$의 $v=(a,b)$ ($a,b$ 복소)로 $\langle v,v\rangle = a\bar a + b\bar b$... 방향이 아니라 **네 규약**($\bar a a + \bar b b$)으로 성분 계산해 허수부가 정말 0임을 확인하라.

(b) 따라서 $\mathrm{Im}\langle\cdot,\cdot\rangle$ 는 (실쌍선형으로서) 같은 벡터 두 개에 0 — **alternating**이다. 이것이 미분 2-형식의 정의 조건임을 한 문장으로 써라. (2-형식의 정의: 같은 것 두 개 넣으면 0.)

(c) 빈칸 채우기로 마무리: "Fubini–Study **폼**은 허미션 내적의 ___에 붙는 이름이고, QFIM은 그 ___이다. 두 얼굴의 계수 데이터는 ___."

### [★] A1-2. 허수부 계수 대조 — 5b의 그 수와 H4의 ω_FS가 같은 수인가

5b에서 $Q_{12} = -\tfrac{i}{4}\sin\theta_2$, $-2\,\mathrm{Im}\,Q_{12} = \tfrac12\sin\theta_2$ 였다. 이제 반대쪽에서: H4의 $\omega_{FS}$를 $w = \tan\tfrac{\theta_2}{2}\,e^{i\theta_1}$ 로 당겨 $(\theta_1,\theta_2)$ 좌표의 계수를 계산하라.

검산 기준: 크기는 $\tfrac12\sin\theta_2$ 로 일치한다 [사전 확인됨]. **부호는 저절로 일치하지 않는다** — $d\theta_1\wedge d\theta_2$ 순서, $\omega_{FS}$ 정의의 $i$, $Q$의 첫 인자 켤레 규약 중 무엇이 부호를 뒤집는지 추적해서 한 줄로 정리하라. (이 추적이 과제의 절반이다.)

### [W] A1-3. 한 점에서 두 얼굴 — 행렬 하나의 분해

$\theta_1=\tfrac\pi2,\ \theta_2=\tfrac\pi3$ 에서 $Q_{ij}=\langle(\partial_i\psi)_\perp,(\partial_j\psi)_\perp\rangle$ 의 $2\times2$ 행렬 전체를 숫자로 써라. 그다음 실수부 행렬과 허수부 행렬로 쪼개고 각각에 이름을 붙여라: 대칭인 쪽 = ___ (계량, $H/4$), 반대칭인 쪽 = ___ (2-형식 계수).

검산 기준: 실수부 $= \mathrm{diag}(\tfrac{3}{16}, \tfrac14)$, 허수부 $= \begin{pmatrix}0 & -\tfrac{\sqrt3}{8}\\ \tfrac{\sqrt3}{8} & 0\end{pmatrix}$.
