# 00 — 연결 지도 (어디서 시작해, 어떻게 이어지나)

> 헷갈릴 때 여기로. 손계산본 `0`–`4` 가 *어디서 시작해 어떻게 한 줄기로 이어지는지* 의 지도다.

---

## 씨앗은 문서 `0` (구면) 하나

모든 것은 **문서 `0`: 구면 $S^2=\mathbb{CP}^1\subset\mathbb R^3$** 에서 갈라져 나온다. 구면 하나를 손으로 다루면 **세 가지**가 한꺼번에 나오고, 그 셋이 각각 `1·2·3` 으로 뻗는다:

```
                          ┌─ (도구) 곡률공식 K=-1/(2λ)Δlogλ ───────────→  문서 1
   문서 0: 구면 ⊂ ℝ³  ────┼─ (임베딩) 구면 점 = 단위 법선 P(x,y) ────────→  문서 2
   (입체사영·제1·2기본형식)  └─ (방법) Gauss 방정식 K=(LN−M²)/(EG−F²) ──→  문서 3
```

- **도구**: 곡률을 등온인자 $\lambda$ 로 재는 공식 → 문서 `1` 이 이걸 거꾸로 쓴다.
- **임베딩**: 점을 $\mathbb R^3$ 의 구체적 점(=단위 법선)으로 적음 → 문서 `2` 의 사영자 $P=vv^*/N$ 이 *바로 그 점을 행렬로 적은 것*.
- **방법**: 제2기본형식으로 곡률을 잼 → 문서 `3` 이 같은 Gauss 방정식을 (높은 차원에서) 그대로 쓴다.

## 줄기 — 한 걸음씩 (`0 → 1 → 2 → 3 → 4`)

| | 한 일 | 다음으로 넘기는 것 |
|---|---|---|
| `0` | 구면 $\subset\mathbb R^3$ → 곡률 $K=1$ (도구·임베딩·방법을 다 얻음) | ↓ *도구를 거꾸로* |
| `1` | "$K=1$"만 부과 → Liouville → **둥근 계량이 유일** | ↓ *그 계량을 복소·사영자로* |
| `2` | 둥근 계량을 **복소화**($g_{z\bar z}$ 한 성분) → 그 계량은 `0`의 구면 점 $=$ **사영자 $P$** 의 유도계량 → **CPⁿ** | ↓ *그 임베딩을 두 번 미분* |
| `3` | CPⁿ 계량의 곡률을 `0`의 **Gauss 방정식 그대로**(제2기본형식) → 모든 정칙 방향 $K=2$ | ↓ *곡률을 적분* |
| `4` | $\int K\,dA=2\pi\chi$ (Gauss–Bonnet) | ↓ *같은 기계를 rank-2로* |
| `6` | 광선(rank-1) → **2-평면(rank-2)**: Gr(2,4), 플뤼커, $K=\det(I{+}Z^\dagger Z)$, $\mathrm{Ric}=4g$, 부피=차수 2 | — |

---

## 각 화살표의 연결고리 (구체적으로 어디서 무엇이)

- **`0 → 1`**: 문서 `1` 은 문서 `0` §5의 공식 $K=-\tfrac1{2\lambda}\Delta\log\lambda$ 에 $K=1$ 을 넣어 Liouville을 푼다.
- **`1 → 2`**: 문서 `2` §1 은 문서 `1` 의 둥근 계량 $\lambda|dz|^2$ 을 *받아* 복소로 적는다($g_{z\bar z}$).
- **`0 → 2`**: 문서 `2` §2 의 사영자 $P=vv^*/N$ 은 문서 `0` 의 구면 점 $\tfrac12(I+\vec P\cdot\vec\sigma)$ 를 행렬로 적은 것 — *같은 임베딩*.
- **`2 → 3`**: 문서 `3` 은 문서 `2` 의 임베딩 $P$ 를 **두 번 미분**(제2기본형식)한다.
- **`0 → 3`**: 문서 `3` 의 Gauss 방정식은 문서 `0` 의 곡면 Gauss $K=\tfrac{LN-M^2}{EG-F^2}$ 의 **일반판**(여차원만 높음).
- **`3 → 4`**: 문서 `4` 는 문서 `3` 의 곡률 $K$ 를 적분한다.

## 걸음 `6` — 같은 기계를 한 층 위로 (Gr(2,4))

걸음 `2`가 "점 = rank-1 사영자"였다면, 걸음 `6`은 **"점 = rank-2 사영자"**다. 새 이론이 아니라 **같은 기계에 입력만 바꾼 것**임을 확인하는 것이 목표:

| 걸음 `0`–`4`에서 한 것 | 걸음 `6`의 대응 | 어디 |
|---|---|---|
| 광선 $[\psi]$, 차트 $(1,z)$ | 2-평면, 차트 $(I\mid Z)$ (가우스 소거) | `6a` §1 |
| — | 소행렬식이 좌표가 되는 관찰 (외적의 일반화) | `6a` §2–3 |
| $K=1+\|z\|^2$ | $K=\det(I+Z^\dagger Z)$ (코시–비네 = 넓이²) | `6b` §2 |
| $g=1/K^2$ ($N=1$) | $g=(I{+}ZZ^\dagger)^{-\mathsf T}\otimes(I{+}Z^\dagger Z)^{-1}$ | `6b` §4–6 |
| $\det g=K^{-2}$, $\mathrm{Ric}=2g$ | $\det g=K^{-4}$, $\mathrm{Ric}=4g$ (일반: $K^{-n}$, $n\,g$) | `6b` §7 |
| $\int K\,dA=2\pi\chi$ (`4`) | $\mathrm{Vol}=\deg\cdot\pi^4/4!=\pi^4/12$ (차수 2 = 직선 4개와 만나는 직선 2개) | `6c` §1 |
| Hopf $S^3\to S^2$, Reeb (파트 IV) | $\Sigma^9\to\mathrm{Gr}(2,4)$, 같은 $\alpha$ | `6c` §3 |
| $\oint d\theta=2\pi$ (원 위의 각도) | **퍼텐셜은 차트마다·그 미분은 전역·안 붙는 정도=감김수**; $\int_{S^2}\omega=\pi$의 정체가 감김수 1 | **`6f`** |
| $\int K\,dA=2\pi\chi$의 "적분이 정수를 안다" | 올적분 / 거듭제곱 / 푸앵카레 쌍대를 갈라 세움; $\alpha$ = 내려오지 못한 퍼텐셜 | `6e` |

문서: `6_…착륙.md`(지도) → `6a`(그라스만·플뤼커) → **`6ab`(퍼텐셜은 공짜가 아니다)** → `6b`(퍼텐셜→계량) → `6c`(부피·길 1·다음 질문) → **`6f`(원 하나로 전부 설명하기)** → `6e`(같은 이야기의 학술 명칭) → `6d`(손계산 0–11 풀이집). 대상 원문 `Gr24_플뤼커당김_계산노트.html`은 외부 노트라 저장소에 없다(문서가 자립적으로 읽히도록 인용을 옮겨 적었다). 검산은 `verify6_plucker.py` (58/58). 합본: `걸음6_Gr24_플뤼커_완전판.html`.

> **`6f`가 왜 `6e`보다 먼저인가.** `6e`는 $S^{11}$·Gysin·오일러류로 답하는데 그건 **이름 붙이기지 설명이 아니다.** 걸음 6의 구조 전체가 **원 위의 각도** 하나로 설명된다: $\theta$는 함수가 아니고($\oint d\theta=2\pi\neq0$), $d$가 상수를 죽여서 $d\theta$는 전역이며, 위층 $\mathbb{R}$에서는 $x$가 전역 함수다. 그리고 호프에서 $\int_{S^2}\omega=\pi$의 $\pi$가 **전이함수 $e^{-i\theta}$의 감김수 1**임이 계산으로 나온다. 큰 기계는 그 다음에 이름으로만 배우면 된다.

> **`6ab`가 왜 따로 있나.** "$K=\sum|p_{ij}|^2$을 대입하면 켈러 퍼텐셜"은 **틀린 문장**이다. 게이지에서 $K\to|\det g|^2K$이므로 $K$는 Gr 위의 **함수가 아니다** — 함수일 수도 없다(전역 퍼텐셜이 있으면 $\omega$가 완전형식이 되어 $\int_{\mathbb{CP}^1}\omega=0$이어야 하는데 우리는 $\pi$를 잰다). 이 걸음의 위상 결론(부피=차수, Fano 지수)이 전부 이 구분 위에 서 있다. 걸음 2의 $K=1+|z|^2$도 실은 차트 하나 위의 것이었다(손계산 9).
