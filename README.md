## 손계산 걸음 (정리_L0-L6/손계산_걸음별/)

`0` 구면→곡률공식 → `1` Liouville → `2` 복소화·사영자→FS → `3` 곡률·Ricci → `4` Gauss–Bonnet → `5` QFIM 남극사영 진단·완성 → `5b` 손노트 결 그대로의 완성본. 각 걸음에 sympy 검산 스크립트(`verify*.py`)가 붙는다.

```
pip install sympy
python "정리_L0-L6/손계산_걸음별/verify5b_note_completion.py"   # 55/55
```

## 규약 (요지)

- 관찰이 먼저, 정의는 그 다음 — 공식은 예시 위에서 목격한 뒤에 쓴다.
- 모든 줄 검산 (sympy), 값이 깨끗한 점 하나에서 숫자로도 확인.
- factor 지뢰 3개: FS vs QFIM의 4배 · dα = 2dA_FS · Tr(dPdP) = ½⟨dp,dp⟩.
