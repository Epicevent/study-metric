# study metric — CP¹에서 Gr(2,4)까지, 손계산으로

## 손계산 걸음 (정리_L0-L6/손계산_걸음별/)

`0` 구면→곡률공식 → `1` Liouville → `2` 복소화·사영자→FS → `3` 곡률·Ricci → `4` Gauss–Bonnet → `5` QFIM 남극사영 진단·완성 → `5b` 손노트 결 그대로의 완성본 → **`6` Gr(2,4)**(rank-1 사영자 → rank-2: 플뤼커·$\det(I{+}Z^\dagger Z)$·$\mathrm{Ric}=4g$·부피=차수). 각 걸음에 sympy 검산 스크립트(`verify*.py`)가 붙는다.

걸음 `6`은 8부작이다: `6_…착륙.md`(지도 — 노트가 하늘에서 떨어트린 것 ↔ 우리가 이미 한 것) → `6a`(그라스만·플뤼커 바닥부터) → `6ab`(**퍼텐셜은 공짜가 아니다** — $K$는 함수가 아니라 선다발 계량, $U(4)$ 불변성이 $K$를 고른다) → `6b`(퍼텐셜→계량 전면 전개) → `6c`(부피=차수, 길 1, 다음 질문) → `6f`(**원 하나로 전부 설명하기** — $\theta$ vs $d\theta$ 하나로 걸음 6 전체; $\int_{S^2}\omega=\pi$의 정체는 감김수 1) → `6e`(같은 이야기의 학술 명칭: 올적분·푸앵카레 쌍대·Gysin·오일러류) → `6d`(손계산 0–11 풀이집).
