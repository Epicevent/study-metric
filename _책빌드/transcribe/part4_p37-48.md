<!-- PAGE 37 -->

$$\alpha(R_\alpha)$$

를 계산하라.

힌트:

$$dx_r(R_\alpha) = -y_r, \qquad dy_r(R_\alpha) = x_r.$$

결과:

$$\alpha(R_\alpha) = x_1^2 + y_1^2 + x_2^2 + y_2^2 = 1$$

on $S^3$.

$$\alpha = \sum_{r=1}^{2} (x_r dy_r - y_r dx_r)$$

에서

$$d\alpha$$

를 계산하라.

검산용 답:

$$d\alpha = 2\sum_{r=1}^{2} dx_r \wedge dy_r.$$

### 필요한 정의

$$\alpha = \frac{i}{2}\sum_{r=1}^{2}\left(z_r d\overline{z}_r - \overline{z}_r dz_r\right).$$

$\mathbb{C}P^1$의 chart

$$U_0 = \{[z_1 : z_2] : z_1 \neq 0\}$$

에서

$$w = \frac{z_2}{z_1}$$

라고 둔다.

local section은

$$s_0(w) = \frac{(1, w)}{\sqrt{1 + |w|^2}}$$

로 잡는다.

### [★★] H3. $s_0^*\alpha$ 계산하기

H2를 이용해

$$s_0^*\alpha$$

를 계산하라.

검산용 답:

$$s_0^*\alpha = \frac{i}{2}\frac{w d\overline{w} - \overline{w} dw}{1 + |w|^2}.$$

이것을 local connection form이라고 볼 수 있다.

---

### [★★] H4. Curvature 계산하기

$$A = s_0^*\alpha = \frac{i}{2}\frac{w d\overline{w} - \overline{w} dw}{1 + |w|^2}$$

라고 두자.

$$dA$$

를 계산하라.

검산용 답:

$$dA = \frac{i\, dw \wedge d\overline{w}}{(1 + |w|^2)^2}.$$

따라서

$$\omega_{FS} = \frac{i\, dw \wedge d\overline{w}}{(1 + |w|^2)^2}$$

로 두면

$$d\alpha = \pi^*\omega_{FS}.$$

---

### [★] H5. 실좌표로 $\omega_{FS}$ 쓰기

$$w = x + iy$$

라고 하자.

다음을 계산하라.

$$dw \wedge d\overline{w}.$$

검산용 답:

$$dw \wedge d\overline{w} = -2i\, dx \wedge dy.$$

<!-- PAGE 40 -->

따라서

$$i\, dw \wedge d\overline{w} = 2dx \wedge dy.$$

즉

$$\omega_{FS} = \frac{2dx \wedge dy}{(1 + x^2 + y^2)^2}.$$

---

### [★] H6. $\int_{\mathbb{C}P^1} \omega_{FS} = 2\pi$ 계산하기

H5의 식을 이용해

$$\int_{\mathbb{C}} \frac{2dxdy}{(1 + x^2 + y^2)^2}$$

를 극좌표로 계산하라.

즉

$$\int_0^{2\pi}\int_0^{\infty} \frac{2r}{(1 + r^2)^2}\, dr\, d\theta$$

를 계산하라.

검산용 답:

$$\int_{\mathbb{C}P^1} \omega_{FS} = 2\pi.$$

$$d\alpha = \pi^*\Omega$$

$$e(P) = \left[\frac{\Omega}{2\pi}\right]$$

$$\int_{S^2} \frac{\Omega}{2\pi}$$

이다.

### [★★] I1. Hopf bundle의 Euler number 계산하기

$$d\alpha = \pi^*\omega_{FS}$$

$$\int_{S^2} \omega_{FS} = 2\pi$$

를 계산했다.

$$\int_{S^2} \frac{\omega_{FS}}{2\pi}$$

를 계산하라.

$$e(S^3 \to S^2) = 1.$$

$$e_\eta(S^3 \to S^2) = -1.$$

$$|e| = 1$$

이다.

## 3단계. QFIM·Fubini–Study·Bures 핵심 47제

> 원본: 07_qfi_fs_hopf_contact (100제)에서 선별. convention 절 필독. 난이도: [W] 웜업 (1–3줄 대입·확인) · [★] 본운동 (발표 수준 유도) · [★★] 도전 (막히면 원본 앞 번호로 후퇴)

### 0. 정규화 convention

이 노트에서는 물리 bra-ket convention을 쓴다.

$$\langle\psi|\phi\rangle$$

은 오른쪽 ket $|\phi\rangle$에 대해 complex-linear이다.

<!-- PAGE 43 -->

**Assertion 1.** 모든 pure state는 normalize한다.

$$\langle\psi|\psi\rangle = 1.$$

**Assertion 2.** 같은 ray는 같은 quantum state이다.

$$|\psi\rangle \sim e^{i\lambda}|\psi\rangle.$$

**Assertion 3.** 이 phase quotient를 density matrix로 죽인다.

$$\rho_\psi = |\psi\rangle\langle\psi|.$$

**Assertion 4.** 이 노트의 정보기하 metric convention은 다음이다.

$$ds^2_{QFI} = 4ds^2_{Bures}.$$

**Assertion 5.** rank-one projector 위에서는

$$ds^2_{Bures} = ds^2_{FS}, \qquad ds^2_{QFI} = 4ds^2_{FS}.$$

**정규화 경고.** 기존 Hopf/contact 노트의 curvature form은

$$d\alpha = \pi^*\Omega, \qquad \int_{\mathbb{C}P^1} \Omega = 2\pi$$

로 normalize되어 있다. 반면 quantum information에서 흔히 쓰는 Fubini–Study Riemannian metric의 area form은 $\mathbb{C}P^1$에서

$$dA_{FS} = \frac{1}{2}\Omega$$

이다. 따라서

$$d\alpha = 2dA_{FS}.$$

또한 $\mathbb{C}P^1$의 이 FS metric은 round sphere 반지름 $1/2$의 metric이므로

<!-- PAGE 44 -->

$$K = 4, \qquad K dA_{FS} = 2d\alpha.$$

이 factor 2를 놓치면 Hopf Euler number 1과 tangent bundle Euler number 2가 섞인다.
