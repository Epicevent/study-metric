## 4단계. $\mathbb{CP}^5$와 Gr(2,4) Plücker metric 핵심 47제

원본: 08_cp1_to_p5_gr24 (100제)에서 선별. 이 폴더의 최종 목적지. 난이도: [W] 웜업 (1–3줄 대입·확인) · [★] 본운동 (발표 수준 유도) · [★★] 도전 (막히면 원본 앞 번호로 후퇴)

### 0. 정규화 convention

이 노트에서는 homogeneous lift $P \in \mathbb{C}^{n+1} \setminus \{0\}$에 대해

$$ds_{\mathrm{FS}}^2 = \frac{\|P\|^2 \|dP\|^2 - \left|\langle P, dP\rangle\right|^2}{\|P\|^4} \tag{0.1}$$

을 Fubini–Study metric으로 쓴다.

Affine chart

$$[Z_0 : Z_1 : \cdots : Z_n] = [1 : z_1 : \cdots : z_n]$$

에서는

$$S = 1 + \sum_{i=1}^{n} |z_i|^2$$

이고 tangent vector $\xi = (\xi_1, \ldots, \xi_n)$에 대해

$$\|\xi\|_{\mathrm{FS}}^2 = \frac{S \sum_{i=1}^{n} |\xi_i|^2 - \left|\sum_{i=1}^{n} \overline{z_i}\xi_i\right|^2}{S^2}. \tag{0.2}$$

동일한 내용을 Kähler potential로 쓰면

$$\Phi = \log S, \qquad g_{i\bar{j}} = \frac{\partial^2 \Phi}{\partial z_i \, \partial \overline{z_j}} = \frac{S\delta_{ij} - \overline{z_i} z_j}{S^2}. \tag{0.3}$$

<!-- PAGE 62 -->

이 convention에서 $\mathbb{CP}^1$은 반지름 $1/2$인 round sphere와 등거리이다. 반지름 1 sphere convention을 쓰면 metric 전체가 4배 커진다.

### A. $\mathbb{CP}^1$ round metric과 FS metric

#### [W] P001. Sphere equation

$$N(x,y) = \left(\frac{x}{1+s}, \frac{y}{1+s}, \frac{1-s}{2(1+s)}\right), \qquad s = x^2 + y^2$$

가 $S^2_{1/2}$ 위에 있음을 보여라.

**검산.**

$$X^2 + Y^2 + T^2 = \frac{1}{4}.$$

#### [★] P002. $N_x$ 계산

$$N_x$$

를 직접 계산하라.

**검산.**

$$N_x = \left(\frac{1 - x^2 + y^2}{(1+s)^2}, \frac{-2xy}{(1+s)^2}, \frac{-2x}{(1+s)^2}\right).$$

#### [★] P003. $N_y$ 계산

$$N_y$$

를 직접 계산하라.

**검산.**

$$N_y = \left(\frac{-2xy}{(1+s)^2}, \frac{1 + x^2 - y^2}{(1+s)^2}, \frac{-2y}{(1+s)^2}\right).$$

<!-- PAGE 63 -->

#### [★] P004. 첫 번째 metric coefficient

$$\langle N_x, N_x \rangle$$

를 계산하라.

**검산.**

$$\langle N_x, N_x \rangle = \frac{1}{(1+s)^2}.$$

#### [★] P006. Pullback metric

$$N^*(dX^2 + dY^2 + dT^2)$$

를 구하라.

**검산.**

$$N^*(dX^2 + dY^2 + dT^2) = \frac{dx^2 + dy^2}{(1 + x^2 + y^2)^2}.$$

#### [W] P007. Homogeneous FS formula

$$P(z) = (1, z)$$

를 이용해 $\mathbb{CP}^1$의 FS metric을 다시 계산하라.

**검산.**

$$ds_{\mathrm{FS}}^2 = \frac{|dz|^2}{(1 + |z|^2)^2}.$$

#### [W] P008. Origin norm

$z = 0$에서 tangent $\xi \in \mathbb{C}$의 norm을 구하라.

**검산.**

$$\|\xi\|_{\mathrm{FS}} = |\xi|.$$

#### [W] P010. Normalization factor

반지름 1 sphere convention과 이 노트의 convention 차이를 설명하라.

**검산.**

$$ds^2_{R=1} = 4\, ds^2_{\mathrm{here}}.$$

### B. $\mathbb{CP}^2$와 conic embedding

#### [W] P011. $\mathbb{CP}^2$ affine FS formula

$[1 : z : w]$에서 tangent $(\xi, \eta)$의 norm을 쓰라.

**검산.**

$$\|(\xi, \eta)\|^2 = \frac{S(|\xi|^2 + |\eta|^2) - |\overline{z}\xi + \overline{w}\eta|^2}{S^2}.$$

#### [★] P012. Coefficient matrix

$S = 1 + |z|^2 + |w|^2$일 때 $g_{i\bar{j}}$를 쓰라.

**검산.**

$$g_{i\bar{j}} = \frac{S\delta_{ij} - \overline{z_i} z_j}{S^2}.$$

#### [★] P016. Raw conic pullback

$(\nu_2^{\mathrm{raw}})^* ds^2_{\mathbb{CP}^2}$를 구하라.

**검산.**

$$(\nu_2^{\mathrm{raw}})^* ds^2_{\mathbb{CP}^2} = \frac{1 + 4s + s^2}{(1 + s + s^2)^2}|dz|^2.$$

#### [★] P017. Normalized conic norm

$$P(z) = (1, \sqrt{2}z, z^2)$$

에서 $\|P\|^2$를 구하라.

**검산.**

$$\|P\|^2 = (1 + s)^2.$$

### C. $\mathbb{CP}^1 \to \mathbb{CP}^m$, $m = 1, \ldots, 5$

#### [W] P024. $m = 1$

$g_1^{\mathrm{raw}}(s)$를 구하라.

**검산.**

$$g_1^{\mathrm{raw}}(s) = \frac{1}{(1+s)^2}.$$

#### [★] P025. $m = 2$

$g_2^{\mathrm{raw}}(s)$를 구하라.

**검산.**

$$g_2^{\mathrm{raw}}(s) = \frac{1 + 4s + s^2}{(1 + s + s^2)^2}.$$

#### [★] P026. $m = 3$

$g_3^{\mathrm{raw}}(s)$를 구하라.

**검산.**

$$g_3^{\mathrm{raw}}(s) = \frac{1 + 4s + 10s^2 + 4s^3 + s^4}{(1 + s + s^2 + s^3)^2}.$$

#### [★] P027. $m = 4$

$g_4^{\mathrm{raw}}(s)$의 numerator를 구하라.

**검산.**

$$1 + 4s + 10s^2 + 20s^3 + 10s^4 + 4s^5 + s^6.$$

#### [★] P028. $m = 5$

$g_5^{\mathrm{raw}}(s)$의 numerator를 구하라.

### D. $\mathbb{CP}^5$ FS metric

#### [W] P031. $\mathbb{CP}^5$ affine $S$

$[1 : u_1 : u_2 : u_3 : u_4 : u_5]$에서 $S$를 쓰라.

**검산.**

$$S = 1 + \sum_{i=1}^{5} |u_i|^2.$$

<!-- PAGE 66 -->

#### [★] P032. Tangent norm

$\xi = (\xi_1, \ldots, \xi_5)$의 FS norm square를 쓰라.

**검산.**

$$\|\xi\|^2 = \frac{S \sum_i |\xi_i|^2 - \left|\sum_i \overline{u_i}\xi_i\right|^2}{S^2}.$$

#### [W] P033. Origin metric

$u = 0$에서 metric을 계산하라.

**검산.**

$$\|\xi\|^2 = \sum_{i=1}^{5} |\xi_i|^2.$$

#### [W] P038. Potential

$\mathbb{CP}^5$의 FS potential을 쓰라.

**검산.**

$$\Phi = \log\left(1 + \sum_{i=1}^{5} |u_i|^2\right).$$

### E. Plucker coordinate

#### [W] P041. Standard plane

$$u = e_1, \qquad v = e_2$$

의 Plucker coordinate를 구하라.

**검산.**

$$[1 : 0 : 0 : 0 : 0 : 0].$$

#### [W] P042. Diagonal graph

$$u = e_1 + e_3, \qquad v = e_2 + e_4$$

의 Plucker coordinate를 구하라.

**검산.**

$$[1 : 0 : 1 : -1 : 0 : 1].$$

<!-- PAGE 67 -->

#### [W] P043. Numeric Plucker coordinate

$$u = (1, 2, 0, 1), \qquad v = (0, 1, 3, -1)$$

의 Plucker coordinate를 구하라.

**검산.**

$$[1 : 3 : -1 : 6 : -3 : -3].$$

#### [★] P044. Basis change

같은 plane의 basis를 $A \in GL_2$로 바꾸면 Plucker coordinate가 어떻게 변하는가?

**검산.**

$$u' \wedge v' = \det(A)\, u \wedge v.$$

#### [★] P045. Klein quadric relation

$\mathrm{Gr}(2,4) \subset \mathbb{CP}^5$의 방정식을 쓰라.

**검산.**

$$p_{12}p_{34} - p_{13}p_{24} + p_{14}p_{23} = 0.$$

#### [★] P046. Solve one coordinate

$$[1 : 2 : 3 : 4 : 5 : x]$$

가 Klein quadric 위에 있으려면 $x$는?

**검산.**

$$x - 10 + 12 = 0, \qquad x = -2.$$

#### [★] P047. Feasibility

$$[2 : 4 : 6 : 8 : 10 : 12]$$

가 가능한 Plucker point인지 판정하라.

**검산.**

$$2 \cdot 12 - 4 \cdot 10 + 6 \cdot 8 = 32 \neq 0.$$

<!-- PAGE 68 -->

#### [★] P048. $P \wedge P$

$$P = \sum_{i<j} p_{ij} e_i \wedge e_j$$

에 대해 $P \wedge P$를 계산하라.

**검산.**

$$P \wedge P = 2(p_{12}p_{34} - p_{13}p_{24} + p_{14}p_{23}) e_{1234}.$$

#### [★★] P049. Decomposability

nonzero 2-form $P$가 decomposable인 조건을 $P \wedge P$로 설명하라.

**검산.**

$$P = u \wedge v \quad \Rightarrow \quad P \wedge P = 0,$$

그리고 4-차원에서는 nonzero $P \wedge P = 0$이면 decomposable이다.

#### [★] P050. Dimension

Klein quadric의 projective dimension을 구하라.

**검산.**

$$\dim_{\mathbb{C}} \mathrm{Gr}(2,4) = 2(4-2) = 4.$$

### F. $p_{12} \neq 0$ chart

#### [W] P051. Graph frame

$$Z = \begin{pmatrix} a & c \\ b & d \end{pmatrix}$$

에 대한 graph frame $F(Z)$를 쓰라.

**검산.**

$$F(Z) = \begin{pmatrix} 1 & 0 \\ 0 & 1 \\ a & c \\ b & d \end{pmatrix}.$$

#### [★] P052. Chart Plucker coordinate

$F(Z)$의 Plucker coordinate를 구하라.

**검산.**

$$[1 : c : d : -a : -b : ad - bc].$$

#### [W] P053. $Z = I$

$$Z = \begin{pmatrix} 1 & 0 \\ 0 & 1 \end{pmatrix}$$

의 Plucker coordinate를 구하라.

**검산.**

$$[1 : 0 : 1 : -1 : 0 : 1].$$

#### [★] P056. Recover chart coordinates

$p_{12} \neq 0$일 때 $a, b, c, d$를 Plucker coordinate로 복원하라.

**검산.**

$$a = -\frac{p_{23}}{p_{12}}, \quad b = -\frac{p_{24}}{p_{12}}, \quad c = \frac{p_{13}}{p_{12}}, \quad d = \frac{p_{14}}{p_{12}}.$$

#### [W] P058. Determinant

$$\Delta = ad - bc$$

가 $Z$의 determinant임을 확인하라.

**검산.**

$$\Delta = \det Z.$$

#### [★] P059. Plucker norm

$$S = \|P(Z)\|^2$$

를 쓰라.

**검산.**

$$S = 1 + |a|^2 + |b|^2 + |c|^2 + |d|^2 + |ad - bc|^2.$$

#### [★★] P060. Binet–Cauchy

$$S = \det(I + Z^* Z)$$

를 직접 전개하라.

**검산.**

$$\det(I + Z^* Z) = 1 + |a|^2 + |b|^2 + |c|^2 + |d|^2 + |ad - bc|^2.$$

### G. Plucker tangent와 FS pullback

#### [W] P063. At $Z = 0$

$Z = 0$에서 $\delta u$를 구하라.

**검산.**

$$\delta u = (\gamma, \eta, -\alpha, -\beta, 0).$$

#### [★] P064. Metric at $Z = 0$

$Z = 0$에서 $\|H\|^2$를 구하라.

**검산.**

$$\|H\|^2 = |\alpha|^2 + |\beta|^2 + |\gamma|^2 + |\eta|^2.$$

#### [★★] P065. General $A$

일반 $Z$에서 $A = \|\delta u\|^2$를 쓰라.

**검산.**

$$A = |\alpha|^2 + |\beta|^2 + |\gamma|^2 + |\eta|^2 + |\delta\Delta|^2.$$

#### [★★] P066. General $B$

일반 $Z$에서 $B = \langle u, \delta u \rangle$를 쓰라.

**검산.**

$$B = \overline{c}\gamma + \overline{d}\eta + \overline{a}\alpha + \overline{b}\beta + \overline{\Delta}\, \delta\Delta.$$

#### [★★] P067. Pullback metric

$\mathrm{Gr}(2,4)$ metric을 $A, B, S$로 쓰라.

**검산.**

$$\|H\|^2 = \frac{SA - |B|^2}{S^2}.$$

#### [★★] P076. Matrix metric formula

matrix chart에서 metric formula를 쓰라.

**검산.**

$$g_Z(H, K) = \mathrm{Re}\, \mathrm{tr}\left((I + ZZ^*)^{-1} H (I + Z^* Z)^{-1} K^*\right).$$

#### [★★] P084. Grassmannian potential

$\mathrm{Gr}(2,4)$의 Kähler potential을 쓰라.

**검산.**

$$\Phi_{\mathrm{Gr}}(Z) = \log \det(I + Z^* Z).$$

#### [★★] P085. Hessian at zero

$Z = 0$에서

$$\partial_i \partial_{\bar{j}} \Phi_{\mathrm{Gr}}$$

을 계산하라.

**검산.**

$$g_{i\bar{j}}(0) = \delta_{ij}.$$

#### [★★] P088. Horizontal tangent at zero

$Z = 0$에서 horizontal tangent condition을 말하라.

**검산.**

$$Q^* \delta Q = 0.$$

## 5단계. QFIM·Grassmannian·세미나 후속 핵심 38제

원본: 09_qfim_statevector_projector_bloch_grassmannian에서 선별. 세미나 발표 트랙과 직결. 난이도: [W] 웜업 (1–3줄 대입·확인) · [★] 본운동 (발표 수준 유도) · [★★] 도전 (막히면 원본 앞 번호로 후퇴)

### 0. Convention

이 노트의 convention은 교수님 노트를 따른다.

**Assertion 1.** 모든 순수상태는 normalize한다: $\psi^* \psi = 1$.

**Assertion 2.** $\psi \sim e^{i\alpha}\psi$는 같은 양자상태이다.

**Assertion 3.** 순수상태 QFIM은

$$H_{ij} = 4\, \mathrm{Re}\left[(\partial_i \psi)^*(\partial_j \psi) - (\partial_i \psi)^* \psi \cdot \psi^*(\partial_j \psi)\right].$$

**Assertion 4.** Fubini–Study metric과의 관계는 $H = 4\, g_{FS}$이다. 즉 $g_{FS}$는 round sphere 반지름 $1/2$, $H$는 round sphere 반지름 1의 metric이다.

**Assertion 5.** 한 큐비트 affine coordinate에서

$$z = x + iy, \qquad D = 1 + |z|^2 = 1 + x^2 + y^2, \qquad \psi(z) = D^{-1/2} \begin{pmatrix} 1 \\ z \end{pmatrix}.$$
