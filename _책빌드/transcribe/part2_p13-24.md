### 고정 규칙

- 포앙카레 원판: $ds^2_{\mathbb{D}} = \dfrac{4|dz|^2}{(1-|z|^2)^2}$, $K = -1$.
- 단위구면/$CP^1$: $ds^2_{S^2} = \dfrac{4|dz|^2}{(1+|z|^2)^2}$, $K = 1$, $\int_{S^2} dA = 4\pi$.
- $\tau = \dfrac{1}{4\pi} dA$, 따라서 $\int_{S^2} \tau = 1$.
- embedded torus는 별도 언급이 없으면 $R = 3, r = 1$을 쓴다.

풀이 힌트는 의도적으로 넣지 않았다. 맨 뒤에 검산용 정답만 둔다.

## 0. 전체 convention

Quaternion 단위는 다음 규칙을 따른다.

$$i^2 = j^2 = k^2 = -1,$$

$$ij = k, \qquad jk = i, \qquad ki = j.$$

반대 순서는 부호가 바뀐다.

$$ji = -k, \qquad kj = -i, \qquad ik = -j.$$

단위 quaternion을

$$q = a + bi + cj + dk, \qquad a^2 + b^2 + c^2 + d^2 = 1$$

로 둔다.

순허수 quaternion 공간은

$$\operatorname{Im} \mathbb{H} = \mathbb{R}i \oplus \mathbb{R}j \oplus \mathbb{R}k$$

이고,

$$xi + yj + zk \longleftrightarrow (x, y, z) \in \mathbb{R}^3$$

로 동일시한다.

---

### 필요한 정의

$$q = a + bi + cj + dk, \qquad q^{-1} = a - bi - cj - dk.$$

$$h(q) = qkq^{-1}.$$

---

<!-- PAGE 20 -->

### [W] A0. 곱셈표 완성하기

다음 표를 완성하라.

| $\cdot$ | $i$ | $j$ | $k$ |
|---------|-----|-----|-----|
| $i$ | | | |
| $j$ | | | |
| $k$ | | | |

검산 기준:

$$ij = k, \qquad ji = -k.$$

---

### [W] A1. $qk$, $kq$를 각각 계산하기

$$q = a + bi + cj + dk$$

에 대해 다음을 계산하라.

$$qk, \qquad kq.$$

그다음 두 결과의 차이

$$qk - kq$$

를 구하라.

---

### [W] A2. $qkq^{-1}$를 두 단계로 계산하기

먼저 A1에서 얻은 $qk$를

$$qk = A + Bi + Cj + Dk$$

꼴로 써라.

그다음

<!-- PAGE 21 -->

$$(A + Bi + Cj + Dk)(a - bi - cj - dk)$$

를 계산하라.

최종 답이 순허수 quaternion이어야 한다. 즉 실수부가 0인지 확인하라.

---

### [W] A3. Hopf map 좌표 공식 얻기

A2의 결과로부터

$$h(q) = qkq^{-1}$$

가 다음과 같음을 보여라.

$$h(q) = (2ac + 2bd)i + (-2ab + 2cd)j + (a^2 - b^2 - c^2 + d^2)k.$$

즉 좌표로는

$$h(a, b, c, d) = \left(2ac + 2bd,\ -2ab + 2cd,\ a^2 - b^2 - c^2 + d^2\right).$$

---

### [W] A4. $h(q) \in S^2$ 확인하기

다음을 직접 계산하라.

$$(2ac + 2bd)^2 + (-2ab + 2cd)^2 + (a^2 - b^2 - c^2 + d^2)^2.$$

결과가

$$(a^2 + b^2 + c^2 + d^2)^2$$

가 됨을 보여라.

따라서 $|q| = 1$이면 $h(q) \in S^2$이다.

---

<!-- PAGE 22 -->

### [W] A5. 특수값 계산하기

다음 네 경우를 계산하라.

$$q = 1, \qquad q = i, \qquad q = \frac{1 + j}{\sqrt{2}}, \qquad q = \frac{1 - i}{\sqrt{2}}.$$

각각에 대해 $h(q)$를 구하라.

---

### [W] A6. $h(-q) = h(q)$ 확인하기

다음을 직접 정리하라.

$$h(-q) = (-q)k(-q)^{-1}.$$

$(-q)^{-1}$가 무엇인지 먼저 쓰고, 부호가 어떻게 사라지는지 계산하라.

---

### 필요한 정의

복소 Hopf map은

$$H(z_1, z_2) = \left(2\operatorname{Re}(z_1\overline{z_2}),\ 2\operatorname{Im}(z_1\overline{z_2}),\ |z_1|^2 - |z_2|^2\right)$$

이다.

Quaternion 좌표와 복소좌표는

$$z_1 = a + di, \qquad z_2 = c + bi$$

로 연결한다.

---

<!-- PAGE 23 -->

### [★] B1. 복소 Hopf map이 $S^2$로 감을 확인하기

$$|z_1|^2 + |z_2|^2 = 1$$

이라고 하자.

$$X = 2\operatorname{Re}(z_1\overline{z_2}), \qquad Y = 2\operatorname{Im}(z_1\overline{z_2}), \qquad Z = |z_1|^2 - |z_2|^2$$

라고 둘 때

$$X^2 + Y^2 + Z^2 = 1$$

임을 보여라.

힌트:

$$X^2 + Y^2 = 4|z_1\overline{z_2}|^2 = 4|z_1|^2|z_2|^2.$$

---

### [★] B2. $S^1$-작용에 대해 불변임을 확인하기

$|\lambda| = 1$일 때

$$H(\lambda z_1, \lambda z_2) = H(z_1, z_2)$$

임을 계산하라.

특히 다음 항이 어떻게 변하는지 각각 확인하라.

$$(\lambda z_1)\overline{(\lambda z_2)}, \qquad |\lambda z_1|^2, \qquad |\lambda z_2|^2.$$

---

### [W] B3. 대표점 계산하기

다음 점들에 대해 $H(z_1, z_2)$를 구하라.

$$(1, 0), \qquad (0, 1), \qquad \left(\frac{1}{\sqrt{2}}, \frac{1}{\sqrt{2}}\right), \qquad \left(\frac{1}{\sqrt{2}}, \frac{i}{\sqrt{2}}\right).$$

---

### 필요한 정의

$$H = \{\cos\theta + \sin\theta\, k : \theta \in \mathbb{R}\}.$$

Hopf map은

$$h(q) = qkq^{-1}$$

이다.

---

### [W] C2. $h^{-1}(k)$ 계산하기

$$h(q) = k$$

라고 하자.

그러면

$$qkq^{-1} = k$$

이다. 이것이

$$qk = kq$$

와 동치임을 보이라.

그다음 A1에서 계산한 $qk$, $kq$를 비교해서

$$b = 0, \qquad c = 0$$

이어야 함을 보여라.

따라서

$$q = a + dk, \qquad a^2 + d^2 = 1$$

이고,

$$h^{-1}(k) = H$$

이다.
