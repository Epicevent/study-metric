### [★] P018. $w$-coordinate에서 FS metric 계산하기

P017의 $\psi(w)$에 대해

$$d\psi$$

$$ds_{FS}^2 = \langle d\psi|d\psi\rangle - |\langle\psi|d\psi\rangle|^2$$

$$ds_{FS}^2 = \frac{|dw|^2}{(1+|w|^2)^2}.$$

### [★] P019. $w = x + iy$에서 area form 계산하기

$$dA_{FS} = \frac{dx \wedge dy}{(1+x^2+y^2)^2}.$$

$$\mathrm{Area}_{FS}(\mathbb{C}P^1) = \pi.$$

## Part C. Rank-one density matrix에서 Bures metric 계산하기

### [★] P021. Projector 미분하기

$$P = |\psi\rangle\langle\psi|$$

라고 하자. 다음을 계산하라.

$$dP.$$

검산:

$$dP = |d\psi\rangle\langle\psi| + |\psi\rangle\langle d\psi|.$$

---

### [★] P022. Gauge direction에서 $dP = 0$인지 확인하기

vertical variation

$$d\psi = i\lambda\psi, \qquad \lambda \in \mathbb{R}$$

를 P021에 넣어라. $dP = 0$이 나오는지 확인하라.

---

### [★] P023. Horizontal gauge 조건 잡기

phase를 적당히 바꾸어

$$\langle\psi|d\psi\rangle = 0$$

이 되게 만들 수 있다고 하자. 이 gauge에서

$$\langle d\psi|\psi\rangle = 0$$

도 성립함을 확인하라.

---

<!-- PAGE 51 -->

### [★] P024. Horizontal gauge에서 $dP^2$ 계산하기

P023의 조건 아래에서

$$dP = |d\psi\rangle\langle\psi| + |\psi\rangle\langle d\psi|$$

를 제곱하라. $dP^2$의 trace를 구하라.

검산:

$$\operatorname{Tr}(dP^2) = 2\langle d\psi|d\psi\rangle.$$

---

### [★] P025. Gauge-free 형태로 $\operatorname{Tr}(dP^2)$ 쓰기

P024의 결과를 gauge-free하게 바꾸어

$$\frac{1}{2}\operatorname{Tr}(dP^2) = \langle d\psi|d\psi\rangle - |\langle\psi|d\psi\rangle|^2$$

임을 보이라.

---

### [★] P026. Pure projector에서 Bures metric 공식 확인하기

rank-one projector 위에서는

$$ds_{Bures}^2 = \frac{1}{2}\operatorname{Tr}(dP^2)$$

라고 둘 수 있다. P025와 비교하여

$$ds_{Bures}^2 = ds_{FS}^2$$

임을 확인하라.

---

<!-- PAGE 52 -->

### [★★] P028. Spectral formula에서 FS metric 얻기

P027에서

$$(dP)_{\mu 0} = \langle\mu|d\psi\rangle, \qquad (dP)_{0\mu} = \langle d\psi|\mu\rangle$$

임을 계산하라. 따라서

$$ds_{Bures}^2 = \sum_{\mu=1}^{d-1} |\langle\mu|d\psi\rangle|^2$$

가 되고, 이것이 FS metric임을 보여라.

---

### [W] P030. 1-qubit에서 QFI metric 쓰기

P015의 결과를 사용하여 1-qubit pure state 위에서

$$ds_{QFI}^2$$

를 $\theta, \phi$로 쓰라.

검산:

$$ds_{QFI}^2 = d\theta^2 + \sin^2\theta\, d\phi^2.$$

### [W] P031. Pauli 항등식 확인하기

### [★] P032. $dP$를 Bloch vector로 쓰기

$$P_{\mathbf{n}} = \frac{1}{2}(I + \mathbf{n}\cdot\sigma), \qquad |\mathbf{n}| = 1$$

$$dP = \frac{1}{2}(d\mathbf{n}\cdot\sigma)$$

### [★] P033. $(d\mathbf{n}\cdot\sigma)^2$ 계산하기

$$(d\mathbf{n}\cdot\sigma)^2 = |d\mathbf{n}|^2 I.$$

$$ds_{Bures}^2 = \frac{1}{2}\operatorname{Tr}(dP^2)$$

임을 확인하라.

### [★] P036. $|d\mathbf{n}|^2$를 $\theta, \phi$로 계산하기

에 대해

$$ds_{Bures}^2 = \frac{1}{4}(d\theta^2 + \sin^2\theta\, d\phi^2)$$

### [★] P038. Fidelity를 Bloch vector로 계산하기

두 pure states의 Bloch vectors를 $\mathbf{n}, \mathbf{m}$이라 하자.

$$P_{\mathbf{n}} = \frac{1}{2}(I + \mathbf{n}\cdot\sigma), \qquad P_{\mathbf{m}} = \frac{1}{2}(I + \mathbf{m}\cdot\sigma)$$

에 대해

$$\operatorname{Tr}(P_{\mathbf{n}} P_{\mathbf{m}})$$

를 계산하라.

검산:

$$\operatorname{Tr}(P_{\mathbf{n}} P_{\mathbf{m}}) = \frac{1}{2}(1 + \mathbf{n}\cdot\mathbf{m}).$$

### [★] P041. Mixed qubit density matrix 쓰기

$$\rho = \frac{1}{2}(I + r\mathbf{n}\cdot\sigma), \qquad 0 \le r < 1, \qquad |\mathbf{n}| = 1$$

라고 하자. eigenvalues가

$$\lambda_+ = \frac{1+r}{2}, \qquad \lambda_- \; \frac{1-r}{2}$$

임을 확인하고, 빠진 등호를 올바르게 고쳐 써라.

---

### [★★] P042. Mixed qubit radial Bures metric

$\mathbf{n}$을 고정하고 $r$만 움직인다. 그러면

$$d\rho = \frac{1}{2}dr\, \mathbf{n}\cdot\sigma$$

이다. eigenbasis에서 spectral formula를 적용하여 radial part가

<!-- PAGE 56 -->

$$ds_{Bures,rad}^2 = \frac{1}{4}\frac{dr^2}{1-r^2}$$

임을 계산하라.

---

### [★★] P043. Mixed qubit angular Bures metric

$r$를 고정하고 $\mathbf{n}$만 움직인다. eigenbasis를 $\mathbf{n} = \hat{z}$인 점으로 회전시켜 계산하라. $d\mathbf{n} = (dn_x, dn_y, 0)$일 때

$$d\rho = \frac{r}{2}(dn_x X + dn_y Y)$$

를 spectral formula에 넣어라. 검산:

$$ds_{Bures,ang}^2 = \frac{1}{4}r^2 |d\mathbf{n}|^2.$$

---

### [★★] P044. Mixed qubit 전체 Bures/QFI metric 쓰기

P042–P043을 합쳐서

$$ds_{Bures}^2 = \frac{1}{4}\left(\frac{dr^2}{1-r^2} + r^2 d\Omega_2^2\right)$$

를 얻어라. 따라서 QFI metric을 쓰라.
