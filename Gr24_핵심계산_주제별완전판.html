머리말 이 판에서 바뀐 것

원전(모음집)은 문제를 출처 파일 순서(1–5단계)로 담고, 주제별 네비게이터가 "몇 단계 몇 번"을 가리키는 구조였다. 이 판은 그 네비게이터의 주제 15개를 그대로 장(章)으로 삼고, 가리키던 문제의 원문 전체를 해당 장에 옮겨 심었다. 따라서:

  • 각 장은 볼 것(규약 — 원문 인라인) → 풀 것(문제 원문, 네비게이터의 순서 그대로) → 검산 기준으로 완결된다. 다른 파일을 열 일이 없다.

  • 문제 번호는 원전 그대로, 앞에 출처 단계만 붙였다: ①A01(1단계) ··· ⑤P105(5단계), Ⓕ(병렬). 원전과 1:1 대조 가능하다.

1장 S²의 계량·면적 — stereographic

모든 것의 출발점. 구면을 평면 좌표 하나 $z$로 읽는 법(입체사영)과, 그 좌표에서 계량·넓이가 $\lambda=\frac{2}{1+|z|^2}$, $dA=\frac{4\,dx\wedge dy}{(1+|z|^2)^2}$ 로 읽히는 것을 손에 붙인다. 이 장의 $\lambda$와 $dA$가 이후 모든 장의 재료다.

볼 것 — 이 판에서의 상태
  • ✓ 인라인 ① 상단 고정 규칙 (orientation!) — 아래 인라인
  • ✕ 제외 문서 QFIM노트 §3–6 (제외된 타자 노트) — 이 장의 규약·문제가 해당 내용을 대체
  • ✕ 제외 문서 손노트 1쪽 — 부록 C 사례연구 참조
검산 기준 전체 면적 $4\pi$ · $dz\wedge d\bar z=-2i\,dx\wedge dy$
① 1단계 상단 고정 규칙 (원문)

원본: 02_geodesic_hodge (100제)에서 선별. 규칙은 원본과 동일. 난이도: [W] 웜업 (1–3줄 대입·확인) · [★] 본운동 (발표 수준 유도) · [★★] 도전 (막히면 원본 앞 번호로 후퇴)

고정 규칙

  • 포앙카레 원판: $ds^2_{\mathbb{D}} = \dfrac{4|dz|^2}{(1-|z|^2)^2}$, $K = -1$.
  • 단위구면/$CP^1$: $ds^2_{S^2} = \dfrac{4|dz|^2}{(1+|z|^2)^2}$, $K = 1$, $\int_{S^2} dA = 4\pi$.
  • $\tau = \dfrac{1}{4\pi} dA$, 따라서 $\int_{S^2} \tau = 1$.
  • embedded torus는 별도 언급이 없으면 $R = 3, r = 1$을 쓴다.

풀이 힌트는 의도적으로 넣지 않았다. 맨 뒤에 검산용 정답만 둔다.

Problem 1. Area Form on $S^2$ in Spherical Coordinates

Let

$$S^2 = \{(x, y, z) \in \mathbb{R}^3 : x^2 + y^2 + z^2 = 1\}$$

and use

$$X(\theta, \phi) = (\sin\theta\cos\phi,\ \sin\theta\sin\phi,\ \cos\theta).$$

(a) Compute $X_\theta$ and $X_\phi$.

(b) Compute $g_{\theta\theta}$, $g_{\theta\phi}$, and $g_{\phi\phi}$.

(c) Verify that

$$g = d\theta^2 + \sin^2\theta\, d\phi^2.$$

(d) Compute $\sqrt{\det(g)}$.

(e) Show that

$$dA = \sin\theta\, d\theta \wedge d\phi.$$

(f) Compute $\operatorname{Area}(S^2) = 4\pi$.

LRGLRG 2교수님 리만기하 PDF (원문)

Problem 2. Area Form on $S^2$ in Stereographic Coordinates

Use stereographic coordinates

$$X(u, v) = \left(\frac{2u}{1 + u^2 + v^2},\ \frac{2v}{1 + u^2 + v^2},\ \frac{u^2 + v^2 - 1}{1 + u^2 + v^2}\right).$$

(a) Compute $X_u$ and $X_v$.

(b) Compute $g_{uu}$, $g_{uv}$, and $g_{vv}$.

(c) Verify that

$$g = \frac{4}{(1 + u^2 + v^2)^2}(du^2 + dv^2).$$

(d) Compute

$$dA = \frac{4}{(1 + u^2 + v^2)^2}\, du \wedge dv.$$

(e) Compute the area by converting to polar coordinates.

Basic Formulas

Let $g = g_{ij}\,dx^i dx^j$ be a Riemannian metric. The inverse metric is denoted by $(g^{ij})$. The volume form is

$$d\operatorname{vol}_g = \sqrt{\det(g_{ij})}\, dx^1 \wedge \cdots \wedge dx^n.$$

For a smooth function $f$,

$$(\operatorname{grad} f)^i = g^{ij}\partial_j f.$$

For a vector field $X = X^i \partial_i$,

$$\operatorname{div} X = \frac{1}{\sqrt{\det g}}\,\partial_i\!\left(\sqrt{\det g}\, X^i\right).$$

The Laplace–Beltrami operator is

$$\Delta f = \operatorname{div}(\operatorname{grad} f) = \frac{1}{\sqrt{\det g}}\,\partial_i\!\left(\sqrt{\det g}\, g^{ij}\partial_j f\right).$$

The Christoffel symbols of the Levi–Civita connection are

$$\Gamma^k_{ij} = \frac{1}{2} g^{k\ell}\left(\partial_i g_{j\ell} + \partial_j g_{i\ell} - \partial_\ell g_{ij}\right).$$

The Hessian of $f$ is

$$(\nabla^2 f)_{ij} = \partial_i \partial_j f - \Gamma^k_{ij}\partial_k f.$$

The Riemann curvature tensor is

$$R^\ell{}_{kij} = \partial_i \Gamma^\ell_{jk} - \partial_j \Gamma^\ell_{ik} + \Gamma^m_{jk}\Gamma^\ell_{im} - \Gamma^m_{ik}\Gamma^\ell_{jm}.$$

For differential forms, the Hodge Laplacian is

$$\Delta_H = d\delta + \delta d,$$

where

$$\delta = (-1)^{n(k+1)+1} * d\, *$$

on $k$-forms in dimension $n$. On functions, up to sign convention,

$$\Delta_H f = \delta d f.$$

The Bochner formula for a smooth function is

$$\frac{1}{2}\Delta |\operatorname{grad} f|^2 = |\nabla^2 f|^2 + \langle \operatorname{grad} f, \operatorname{grad} \Delta f\rangle + \operatorname{Ric}(\operatorname{grad} f, \operatorname{grad} f).$$

(b) 그래서 QFIM convention이 $H_{ij} = 2\operatorname{Tr}(\partial_i P\, \partial_j P)$ (factor 2)를 쓰면 정확히 $\langle \partial_i p, \partial_j p \rangle$가 복원됨을 확인하라.

(c) 복소좌표에서 $\operatorname{Tr}(\partial_z P\,\partial_{\bar z}P) = \frac{1}{D^2}$를 계산하고, $ds^2 = 2\cdot 2\operatorname{Tr}(\partial_z P\,\partial_{\bar z}P)\,dz\,d\bar z = \frac{4\,dz\,d\bar z}{D^2}$ 로 $E, F, G$ 조립 없이 바로 metric이 나옴을 확인하라.

이 노트에서는 homogeneous lift $P \in \mathbb{C}^{n+1} \setminus \{0\}$에 대해

$$ds_{\mathrm{FS}}^2 = \frac{\|P\|^2 \|dP\|^2 - \left|\langle P, dP\rangle\right|^2}{\|P\|^4} \tag{0.1}$$

Affine chart

$$[Z_0 : Z_1 : \cdots : Z_n] = [1 : z_1 : \cdots : z_n]$$

$$\|\xi\|_{\mathrm{FS}}^2 = \frac{S \sum_{i=1}^{n} |\xi_i|^2 - \left|\sum_{i=1}^{n} \overline{z_i}\xi_i\right|^2}{S^2}. \tag{0.2}$$

동일한 내용을 Kähler potential로 쓰면

$$\Phi = \log S, \qquad g_{i\bar{j}} = \frac{\partial^2 \Phi}{\partial z_i \, \partial \overline{z_j}} = \frac{S\delta_{ij} - \overline{z_i} z_j}{S^2}. \tag{0.3}$$
